变色方块
========

变色方块是一款休闲益智小游戏，适合所有人群，点击方块会改变方块颜色，将所有方块变为蓝色即为胜利，快快开动你的脑筋，来挑战吧，超有难度哦

## 构建
第一步，安装r.js

```
$ npm install -g requirejs
```

第二步，打包js
```
# 打包web页面js，index.js → main.js
$ npm run build:web

# 打包微信页面js，weixin.js → weixin-main.js
$ npm run build:weixin
```
